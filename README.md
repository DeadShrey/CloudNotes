# CloudNotes

A simple minimalistic note taking website made using Google Antigravity.
> https://gs-cloud-notes.web.app

## Why CloudNotes?

Most note-taking tools feel either:
- Too bare, or
- Too heavy

CloudNotes sits in between:
- Simple like Notepad
- Persistent like the cloud
- Focused on the act of writing, not managing

## Features

- Clean, distraction-free editor
- Automatic saving
- Image support with predictable layout
- Turn-based collaboration via shareable note keys
- Login-based persistence
